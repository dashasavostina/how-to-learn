## Краткое описание
Одностраничный сайт, созданный с использованием HTML и CSS.
Реализован по правилам Flexbox, также используется CSS анимация. Файловая структура организована по файловой структуре Nested БЭМ. 

### Стек технологий
• HTML 5
• CSS 3
